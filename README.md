# Mountain Cabin Rental ‚Äî Cabin Rental Web System

## Overview
**Mountain Cabin Rental** is a web application that enables users to browse, search and reserve mountain cabins in Serbia. It supports multiple user roles with different permissions and tailored interfaces.

There are three primary user types:
- **Tourist** ‚Äì Browses cabins, makes reservations, views personal reservations.
- **Owner** ‚Äì Manages cabin listings, reviews bookings.
- **Administrator** ‚Äì Oversees system-wide data and user accounts.

The platform features secure login, dynamic content display, sortable and searchable lists, and interactive forms for reservations and profile management. It follows modern web development practices to provide a responsive and intuitive user experience.

## üîë Key Features

- **User Authentication**
  - Login form for tourists, owners, and admins.
  - Registration with validation and secure storage of credentials.
  - Password change functionality.

- **Cabin Discovery**
  - Browse all available cabins with details and images.
  - Sort and search by name or location.

- **Reservations Management**
  - Tourists can book cabins for given dates.
  - Owners review, accept or reject bookings.
  - Calendar visualization of reservations.

- **Responsive UI**
  - Works on desktop, tablet, and mobile devices.
  - Consistent layout with navigation menus on all pages.

- **Admin Dashboard**
  - Administrative oversight of users and bookings.
  - Ability to deactivate or update user profiles.
